📝 Petunjuk Setup Lengkap
1️⃣ Setup Fonnte API (WhatsApp)
  1. Buat akun Fonnte
  2. Verifikasi WhatsApp bot
  3. Get API token dari dashboard
  4. Paste token di field "Fonnte API Token" di atas
2️⃣ Setup Google Cloud Project
  1. Buat project di Google Cloud Console
  2. Enable APIs: Google Sheets API + Google Calendar API
  3. Buat Service Account dan download JSON key
  4. Store JSON key di path: JSON GCP/credentials.json
3️⃣ Setup Google Spreadsheet
  1. Buat spreadsheet baru di Google Sheets
  2. Share ke service account email: django-calendar-app@cuti-gcal.iam.gserviceaccount.com
  3. Buat 4 sheets: Pengaturan, TargetPenerimaWA, Meetings, Logs
  4. Copy spreadsheet ID ke Meeting Reminder Settings per departemen
4️⃣ Deploy Google Apps Script
  1. Buka spreadsheet → Tools → Script Editor
  2. Paste GAS code dari MEETING_REMINDER_GAS_CODE.gs
  3. Save dan authorize
  4. Klik menu "2. Atur Trigger Harian" untuk setup daily trigger
